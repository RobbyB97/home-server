Projects
B.A.D.
  • Find B.A.D. here!

    B.A.D. (Bearer of Awesome Data) is a program written in Python that dynamically generates webpages that give the user easy access to the latest news on whatever interests them. While working on this project, I've split some aspects of it that could prove useful to others and deployed them to the Python Package Index.

  • yt-iframe

    yt-iframe is a Python package that takes a link to a YouTube video or channel and gathers useful information about it. It also generates iframes for the video or each video in the channel. If you would like to embed a YouTube video or channel's videos on your website, this could be useful to you. (Run pip install yt-iframe)

  • libpyn

    libpyn is an API for libsyn, a podcast hosting website. If you are hosting a channel on this site and would like to post updates on your personal website automatically, or you are a podcast enthusiast that likes having all their podcasts in one place, or would like to download podcasts and listen to them offline, libpyn can help! (Run pip install libpyn)

WP Plugins

Developing WordPress plugins helps me to learn PHP and the WP API, and helps other people who may not be experienced with coding to create their own website with highly customizable and easy to use plugins.
